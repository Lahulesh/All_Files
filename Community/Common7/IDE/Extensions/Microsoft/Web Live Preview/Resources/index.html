Microsoft Edge DevTools for Visual Studio

Welcome to the Edge DevTools hosted inside Visual Studio!

You can now use the Edge Developer Tools inside your editor to inspect, change and debug your Web Live Preview using the Elements and Network tool.

Simply open a Web Live Preview instance for your file and connect using the attach button in the toolbar above.

If there is a Web Live Preview instance active and the attach button does not attach the DevTools, please check your internet connection.

If you have any questions or concerns, please check out our Edge DevTools for VS Code GitHub repository, read up on how to use the extension or report us your problems and ideas. We're happy to listen!

Although the GitHub repo focuses on our VS Code extension, most of the information there can apply to the Visual Studio extension as well.
